About This Website

This virtual museum was created by Hin Lam Tsang as part of a Digital Humanities and Global History project. It explores how East and West Germany prosecuted Nazi crimes after World War II, focusing on key trials such as the Frankfurt Auschwitz Trial and the Horst Fischer Trial. The project combines legal history, political context, and public memory, using maps, media, and archival material to make the story accessible.

All content is presented for educational and non-commercial purposes. Images and data sources are credited where applicable. This website is hosted using GitHub Pages and built using HTML, CSS, and JavaScript.

If you have questions, comments, or suggestions, please contact the creator via the GitHub repository.
